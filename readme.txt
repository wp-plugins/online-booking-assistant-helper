An assistant plugin for the website service Online Booking Assistant, which can be found at http://www.onlinebookingassistant.com

== Description ==

This plugin allows you to copy and paste the code given in the OBA users control panel, into the settings page, which will then give you unique short codes that you can use to show the contact forms within any post, page or widget on the wordpress website.

The 5 shortcodes that will work once the settings page has been completed are:

[OBAFirstContactSmall] - for the small width First Contact Form
[OBAFirstContactMedium] - for the medium width First Contact Form
[OBAFirstContactLarge] - for the large width First Contact Form
[OBAReserveADate] - for the Reserve A Date Form
[OBAFeedback] - for the Feedback Form

The shortcodes will be replaced with the code from OBA when the pages containing those codes are requested by your visitors.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the `online_booking_assistant_helper` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Copy and paste the code given in the Settings > Forms page of the OBA control panel (http://my.onlinebookingassistant.com/settings-forms.html) into the settings page of the plugin
4. Place the following shortcodes on any page, post or widget of your website: [OBAFirstContactSmall] [OBAFirstContactMedium] [OBAFirstContactLarge] [OBAReserveADate] [OBAFeedback]
